## 预渲染技术
什么是预渲染呢？

很好理解，就是当我们还没有访问页面时提前对页面进行渲染，等到我们真正访问页面时就不需要再话费额外的时间去渲染页面了。

在当今网页最重要的性能指标(Core Web Vitals)中,Largest Contentful Pint(LCP)(最大内容渲染)占据着重要的位置，这个指标也很好理解，也就是一个网页当前视口中可见的最大元素的渲染时间。
