- 文生图关键词公式
    
    内容描述+ 风格说明+ 构图镜头+ 属性设定

    内容描述: 你想呈现的画面 人/物、动作、低点等，例如：一个女孩子正拿着一本书阅读

    风格说明：你需要的风格 插画、游戏、UI设计、照片、漫画、二次元、3D、VR、AR等,例如迪士尼动画

    构图镜头: 你想要的构图/镜头形式 全身、半身、特点、头像、广角等，例如：Blender3d,人设图

    属性设定：后缀词，质量、尺寸、版本，例如长宽比
