## tig
tig 的几个主要用法

1. 显示 repo 的更新日志：tig log

2. git 文件中使用正则匹配搜索 tig grep config

3. 显示 git repo 的状态： tig status

更多其他 tig 命令：

tig -h


[如何使用 Tig 浏览 Git 日志](https://linux.cn/article-11069-1.html)
